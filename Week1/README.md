### Deskripsi
Tugas pertemuan 1 membuat aplikasi bertujuan untuk menampilkan Inputan user dengan menekan tombol, dan akan muncul HALO "Inputan",
dan pada aplikasi juga bisa mengatut tema gelap atau terang, berikut link Youtube presentasi aplikasi



Fitur aplikasi:
- Menampilkan Inputan dengan menekan tombol
- Mengatur Tema

### Cara Menjalankan

1. **Clone repository** ini ke komputer Anda:
   ```bash
   git clone [https://github.com/MuhammadNafisPutra/PemogramanMobile.git]
2. Buka Android Studio, lalu pilih Open.
3. Cari dan pilih folder PemogramanMobile yang baru saja di-clone.
4. Tunggu hingga proses Gradle Sync selesai (pastikan koneksi internet lancar).
5. Jalankan aplikasi
